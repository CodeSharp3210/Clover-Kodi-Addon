# Clover-Kodi-Addon
A Kodi Addon to show other your Kodi activity in Discord with other tools like Cache and Logs Cleaner. Developed By CodeSharp

To use the addon for RPC, you need to follow these steps:

WARNING: To use this addon, you must be on the same connection between the Kodi Client and the Discord bridge server.


1) Download Python and open a terminal on Windows or Linux, then run this command: > pip install pypresence
Next, also run this command: > pip install flask3

2) Copy the file discord_bridge.py to a conveniently accessible directory.

3) Run discord_bridge.py on your computer with Discord open.

4) Open Kodi and install the addon by .zip method

5) Enable the addon through Kodi's settings, and if you have correctly followed these steps, you should see requests coming in every 2 seconds in the discord_bridge.py terminal. This is normal, donâ€™t worry. It means the addon is working correctly.
